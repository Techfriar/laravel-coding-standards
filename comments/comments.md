## Comments

    - Single-line Comments  :           
                            - // This is a single-line comment

    - Multi-line Comments   :            
                            - /*
                               This is a multi-line comment.
                               It can span multiple lines and is useful for longer explanations.
                              */

    - blade 	            :     
                            - {{-- This is a Blade comment. --}}
	
    - html element	        :        
                            -  <!-- This is an HTML comment. -->
